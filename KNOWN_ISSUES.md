# 알려진 문제점 (Known Issues)

## 🔍 Web Application (10_webapp) 항목 상세 내용 부족

### 문제 설명

Web Application 챕터의 일부 항목에서 "점검 및 조치 사례" 섹션의 상세 내용이 누락되었습니다.

### 영향을 받는 항목

- **CI (코드 인젝션)**: LDAP, SSI, XPath, XXE, SSTI 인젝션 등 각 세부 항목의 테이블과 예제 누락
- **SI (SQL 인젝션)**: 상세 점검 방법 일부 누락
- 기타 Web Application 항목들

### 현재 상태

```markdown
## 점검 및 조치 사례

### LDAP 인젝션

1. **사용자 입력값에 대하여 변조된 LDAP 쿼리 삽입 후 실행 가능 여부 확인**

####  LDAP 쿼리 삽입 

# ❌ 이 아래에 테이블과 상세 예제가 있어야 함
```

### 원인

초기 PDF 파싱 과정에서 Web Application 챕터의 복잡한 테이블 구조가 완전히 추출되지 않았습니다.

### 해결 방법

#### 옵션 1: 원본 PDF 참조
현재는 **KISA 원본 PDF 문서**를 함께 참조하시기 바랍니다.
- 페이지: 761~781 (CI~WM)
- 원본에는 각 항목별 상세 테이블과 예제 포함

#### 옵션 2: 향후 개선
Web Application 챕터만 선별하여 재파싱 필요

### 정상 작동하는 부분

- ✅ 모든 항목의 개요, 점검 내용, 점검 목적
- ✅ 보안 위협, 판단 기준, 조치 방법
- ✅ 항목 간 네비게이션 및 검색
- ✅ 다른 11개 챕터는 완벽하게 작동

## 💡 권장 사항

Web Application 취약점 진단 시:
1. 이 MkDocs 사이트에서 개요와 기본 정보 확인
2. 상세 점검 방법은 원본 PDF 참조
3. 두 문서를 병행 사용
